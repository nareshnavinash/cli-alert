## Usage Tips

- Attribution: `-- shelldone` or link to `https://github.com/nareshnavinash/shelldone`
- Hashtags: `#shelldone #terminal #cli #devtools #bash #notifications`
- Pair with: a screenshot of a notification, a demo GIF, or the landing page URL
- Best format on Twitter/X: quote in monospace (wrap in backticks) + one-line description + link
- Best format on Reddit: quote as title or opening line, then describe the tool
